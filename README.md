Descripción del proyecto 

Aplicación web sencilla y práctica para gestionar una lista de supermercado. 
Permite agregar productos, eliminarlos individualmente, realizar la compra o cancelarla, mostrando mensajes dinámicos y actualizando el contador en tiempo real. 
El proyecto está pensado como ejercicio académico y de práctica en desarrollo web moderno. 

Funcionalidades implementadas 

•	Agregar productos: mediante un cuadro de texto y botón. 
•	Eliminar productos individuales con un botón al lado de cada ítem. 
•	Comprar: vacía la lista, reinicia el contador y muestra mensaje de confirmación. 
•	Cancelar compra: vacía la lista, reinicia el contador y muestra mensaje de cancelación. 
•	Mensajes temporales que desaparecen automáticamente después de 1 segundo. 
•	Persistencia con localStorage: los productos se mantienen incluso al recargar la página. 
•	Contador dinámico: que muestra el total de productos agregados. 

Tecnologías usadas 

•	HTML5: estructura semántica del proyecto. 
•	CSS3: estilos modernos, uso de Flexbox para organizar elementos. 
•	JavaScript: lógica de interacción, manipulación del DOM y gestión de eventos. 
•	localStorage: almacenamiento persistente de productos en el navegador. 

Uso de la IA en el proyecto 

localStorage: se implementó con apoyo de IA para garantizar persistencia de datos y buena práctica en la gestión de arrays y objetos. 

Flexbox: se aplicó con sugerencias de IA para organizar la lista y los botones de manera moderna y responsiva. 

Depuración y mejoras: la IA asistió en la corrección de errores de sintaxis, duplicados de `id` y optimización de funciones (`guardar () `, `renderLista () `). 

Cómo usar 

1.	Escribe un producto en el cuadro de texto y presiona Agregar 
2.	El producto aparecerá en la lista con un botón Eliminar. 
3.	Usa Comprar para vaciar la lista y confirmar la compra. 
4.	Usa Cancelar compra para vaciar la lista y mostrar mensaje de cancelación. 
5.	El contador se actualiza automáticamente y los datos se guardan en el navegador gracias a localStorage.
